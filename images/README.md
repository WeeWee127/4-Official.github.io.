# Папка для зображень

Тут зберігайте всі фотографії для сайту:

## Структура:

### `/images/team/` - Фото команди
- `team-photo.jpg` - головне фото команди (для секції "О команде")

### `/images/players/` - Аватари гравців
- `player1.jpg` - аватар гравця 1
- `player2.jpg` - аватар гравця 2
- `player3.jpg` - аватар гравця 3
- `player4.jpg` - аватар гравця 4
- `player5.jpg` - аватар гравця 5

### `/images/sponsors/` - Логотипи спонсорів
- `sponsor1.png` - логотип спонсора 1
- `sponsor2.png` - логотип спонсора 2
- `sponsor3.png` - логотип спонсора 3
- `sponsor4.png` - логотип спонсора 4

## Рекомендації:
- Формат: JPG для фотографій, PNG для логотипів
- Розмір фото команди: 800x600px або більше
- Розмір аватара гравця: 400x400px (квадратне)
- Розмір логотипу спонсора: 300x150px або пропорційно
